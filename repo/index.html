core beliefs

三个判断，决定接下来三年怎么走

// 01
应用层，才是接下来的主战场
底层模型正在商品化——GPT-4 级别的能力开始成为基础设施。 真正的差异化在应用层。

iPhone 出来之后，改变世界的是 App Store，不是 ARM 芯片。 Agent 经济的逻辑是一样的：谁定义分发平台，谁掌握生态入口。

Agent Store、Skill Store，这些东西现在还没有主导者。窗口是开着的。
// 02
Agent 就是 Web2 到 Web3 最自然的桥梁
公司战略要做 Web2 到 Web3 的通道。Agent 就是那个通道—— 用户根本不需要理解区块链，就能通过 Agent 使用链上服务。

这是用最短路径实现公司最核心的战略目标。

AICY 的交易 Agent 已经绑定了 ERC-8004。 这是白皮书 §3 账户模型在应用层的第一个具体实现。
// 03
先把收入做出来，再去谈大的规划
机器人数据平台、SuperApp 这些大方向我都认， 但那是 18-24 个月之后的故事。

Agent 平台可以在数周内产生第一笔收入： 交易手续费、对战结算费、订阅制——边际成本接近零。

基础要先打牢，收入要先做出来。
